## Setup

### Run the backend server :
1. Install python and pip
2. Open terminal / command prompt
2. go to folder "src"
3. pip install -r .\requirements.txt
4. python .\main.py

### Run the frontend server :
1. Install Node.js
2. go to folder "src/frontend"
3. npm install
4. npm start

### Opening the web app :
1. open browser
2. go to localhost:3000/
